# v0.1.0-alpha.1

First public alpha for Guild AgentDesk.

## Install

```bash
go install github.com/lucid-fdn/guild/cli/cmd/guild@v0.1.0-alpha.1
guild agentdesk init
guild agentdesk doctor
guild mcp serve
```

## What This Alpha Proves

- GitHub Issues labeled `agent:ready` can become local mandates.
- Agents can claim work so multiple agents do not silently take the same task.
- Guardrails, bounded context, approvals, proof, handoff, verification, and replay are available without a hosted service.
- The single binary exposes AgentDesk over MCP with `guild mcp serve`.
- `guild agentdesk doctor` checks config, directories, GitHub label readiness, MCP availability, and proof readiness.
- CI can run `guild agentdesk verify --github-report` and post an Agent Work Contract report on PRs.

This alpha intentionally uses the Go binary as the primary install path.
The TypeScript MCP adapter remains in-repo for adapter development and host compatibility testing; npm publishing waits until the package scope and name are final.
