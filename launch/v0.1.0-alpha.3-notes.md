# v0.1.0-alpha.3

Third public alpha focused on removing first-run workflow noise and improving the copy-paste demo.

## Install

```bash
go install github.com/lucid-fdn/guild/cli/cmd/guild@v0.1.0-alpha.3
guild agentdesk init
guild mcp serve
```

## New

- Generated workflows disable Go dependency caching so non-Go repos do not show noisy `go.sum` cache warnings.
- README and quickstart include a copy-paste external demo path from bootstrap to replay.
- `guild agentdesk bootstrap github --repo owner/repo` remains the first-run GitHub setup path.

## Copy-Paste Demo

Use this release from a clean external repo:

```bash
GITHUB_TOKEN="$(gh auth token)" \
guild agentdesk bootstrap github --repo lucid-fdn/<repo>
```

Then create an issue labeled `agent:ready`, run `guild agentdesk next --source github`, claim it, attach proof, verify, and open a PR.
