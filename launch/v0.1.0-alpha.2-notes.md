# v0.1.0-alpha.2

Second public alpha focused on first-run GitHub DX.

## Install

```bash
go install github.com/lucid-fdn/guild/cli/cmd/guild@v0.1.0-alpha.2
guild agentdesk init
guild mcp serve
```

## New

- `guild agentdesk bootstrap github --repo owner/repo`
- Portable GitHub Actions workflow generation for repos that do not contain Guild source.
- Agent-ready GitHub issue template generation.
- AgentDesk doctor workflow generation.
- GitHub label creation for `agent:ready` and priority labels when `GITHUB_TOKEN` is set.

## Dogfood Goal

Use this release from a clean external repo:

```bash
GITHUB_TOKEN="$(gh auth token)" \
guild agentdesk bootstrap github --repo lucid-fdn/<repo>
```

Then create an issue labeled `agent:ready`, run `guild agentdesk next --source github`, claim it, attach proof, verify, and open a PR.
