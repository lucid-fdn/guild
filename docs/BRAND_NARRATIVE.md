# Brand Narrative

## Working Name

`Polity`

Status: long-term storytelling spine, not the current launch positioning.
The current product wedge is agent-first: every agent run starts with a mandate and ends with proof.

Polity means an organized community with governance, roles, rules, and institutional memory.

That is the product thesis:

```text
Agents do not scale as swarms.
They scale as institutions.
```

The project should not sound like another agent framework.
It should sound like the missing civic layer above agent frameworks.

## Core Story

The next jump in AI will not come from simply spawning more agents.

Humans do not coordinate at scale because everyone talks to everyone.
Humans coordinate at scale because we invented institutions:

- mandates
- offices
- records
- consent
- courts
- laws
- shared culture

Agent teams need the same thing.

Polity gives any orchestrator the institutional primitives that make many agents accountable, inspectable, and capable of improving together.

## Mapping

| Product primitive | Civilization metaphor |
| --- | --- |
| `Taskpack` | mandate |
| `DRI` | accountable citizen / office holder |
| `Artifact` | public record |
| `Replay` | institutional memory / court record |
| `Approval` | consent |
| `Promotion Gate` | law |
| `Commons` | shared culture |

This is the storytelling spine.

`Taskpack` is not just a task format. It is a mandate: bounded authority, context, constraints, and acceptance criteria.

`DRI` is not just assignment. It is office: one accountable holder of responsibility.

`Artifact` is not just output. It is the public record: durable, attributable, and inspectable.

`Replay` is not just debugging. It is institutional memory: a court record the institution can review later.

`Approval` is not just a modal. It is consent: a human grants authority when risk crosses a boundary.

`Promotion Gate` is not just evaluation. It is law: the rule that decides what may enter the institution.

`Commons` is not just memory. It is shared culture: the accepted patterns the institution teaches its future agents.

## Why Not Matrix Branding

Matrix-inspired names are memorable, but they pull the product toward simulation, rebellion, and pop-culture IP.

That is the wrong center of gravity.

Polity should feel like infrastructure for serious teams:

- open-source
- inspectable
- accountable
- governance-native
- enterprise-legible

The useful Matrix-adjacent concept is internal language:

```text
Construct = the replay/simulation environment.
```

Possible phrase:

```text
Run it in the Construct before it enters the Commons.
```

That can be product flavor later, not the main brand.
